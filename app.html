App Portfolio

Chatly

Chatly is a modern chat application fully built in Flutter. It offers real-time messaging with a clean and responsive UI. Users can send text, images, and manage conversations seamlessly. The app is optimized for both Android and iOS with smooth performance. Chatly demonstrates powerful Flutter + backend integration for a complete messaging experience.

Prodigeniues

I created the Prodigenius app, using TensorFlow Lite to power AI-driven task prioritization and productivity tracking. The app analyzes task urgency, due dates, and preferences, offering smart recommendations to help users stay organized and efficient. TensorFlow Lite ensures fast processing, providing a smooth user experience.

Independence Care

This app is built with Flutter and seamlessly integrated with a dynamic website via APIs. Caregivers can log in securely to access their assigned duties and schedules. The app enables clock in/out functionality with real-time location tracking. Administrators can monitor attendance and activity directly through the connected web dashboard. Designed for efficiency, it simplifies work management and accountability for caregivers and organizations.

Tapxi Driver

Tapxi Driver is a driver-focused app enabling easy ride request management. Drivers receive ride notifications and can accept or reject trips seamlessly. The app includes features like earnings tracking, trip details, and navigation support. Optimized for performance, it ensures drivers stay connected in real-time. Tapxi Driver empowers drivers to maximize efficiency and earnings on the platform.

Tapxi Passanger

Tapxi Passenger is a ride-booking app built in Flutter for a smooth travel experience. Users can book rides instantly and track drivers in real-time on the map. The app provides fare estimates, trip history, and secure payment options. A clean and intuitive UI ensures effortless navigation for all passengers. Designed for reliability, Tapxi Passenger makes ride-hailing fast and convenient.

Yatra Facilitiation

Yatra Facilitation is a Flutter-based app designed for Sikh and Hindu pilgrims visiting Pakistan. It helps users easily discover and navigate sacred places and Gurdwaras/temples across the country. The app provides historical and cultural details about each holy site. An integrated AI-powered guide assists travelers with personalized recommendations. With a simple and intuitive UI, Yatra Facilitation makes spiritual journeys easier, informative, and memorable.

Expense Tracker

This Flutter app helps users track expenses with a responsive layout, chart, and list of individual expenses. Add and delete expenses using a bottom sheet, with undo option. The app uses shared preferences to save and expenses locally, aiming for an easy-to- and visually appealing.

Admob Tester

This app is built in Flutter to test and demonstrate Google AdMob integration. It allows developers to quickly verify that ads are working correctly. The UI is clean and responsive, following Flutter’s cross-platform design principles. It supports banner, interstitial, and rewarded ads for full testing coverage. The app is lightweight, efficient, and perfect for developers experimenting with AdMob in Flutter projects.

Calculator

This is a Flutter calculator app with a customizable design. It has a history page to review previous calculations.The calculator has large buttons with clear labels for easy use. The history page keeps a record of all calculations, allowing users to review them at any time. The delete button clears the most recent entry.

Gym Guide Elite

Gym Guide Elite is my debut Play Store app, featuring 7-day exercise plans, customized food plans, and an AI trainer for personalized guidance. It simplifies fitness routines with structured workouts and nutritional support.

Dice Roller

Dice Roller is a Flutter app that simulates the roll of a dice. The app has a simple and user-friendly interface, with a single button to roll the dice. When the button is pressed, the generates a random number between 1 and 6, and displays the corresponding dice image.

Flutter Quiz

This is a Flutter app for a simple quiz game. The start screen displays a "Let's Start" button that, when pressed, takes the user to the question screen. The question screen displays a question and a set of answer options.The app uses a gradient background and a dark theme for an appealing visual design.

Ring Tune App

RingTuneApp is a Flutter-based app that lets users set custom ringtones on their device. Users can browse, play, and preview ringtones with a simple interface. The app allows sharing ringtones easily with friends through the platform. It supports quick ringtone assignment for calls, notifications, and alarms. With smooth performance and user-friendly design, RingTuneApp makes personalizing your phone fun and easy.
